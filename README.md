# Get Started

## Requisitos
- Node >= 16.13.1
- npm >= 8.1.2

## Quick start

Install dependences

```
npm install
```

Launch environment

```
npm run dev
```

Now you can open http://localhost:5050

### Packages included
- [Webpack](https://webpack.js.org/)
- [React Router](https://reactrouter.com/en/main)
- [Material UI](https://mui.com/material-ui/getting-started/usage/)
